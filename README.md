# E-Shop - Modern E-Commerce Platform

A fully functional e-commerce application built with React and Redux Toolkit, featuring a complete shopping cart system, product browsing, and responsive design.

## ‚ú® Features

### Shopping Experience
- Browse products from external API with high-quality images
- View detailed product information
- Search and filter products
- Responsive product grid layout
- Similar products recommendations

### Shopping Cart
- Add products to cart
- Increase/decrease item quantities
- Remove items from cart
- Real-time cart total calculation
- Cart item count badge in navbar

### User Interface
- Clean, modern UI with DaisyUI components
- Fully responsive design (mobile, tablet, desktop)
- Loading states for better UX
- Error handling

## üì¶ Installation & Setup
```bash
# Clone the repository
git clone https://github.com/lezyCoder/e-shop.git

# Navigate to project directory
cd e-shop

# Install dependencies
npm install

# Start development server
npm run dev
```

The app will open at `http://localhost:5173`
